# Hand Cricket 🏏

A terminal-based Hand Cricket game built with Python.

## How to Run
```bash
python hand_cricket.py
```

## How to Play
1. **Toss** - Choose Odd or Even, then enter a number (1-6)
2. **Selection** - Winner chooses to Bat (BA) or Bowl (BO)
3. **Batting** - Enter a number each ball to score runs
4. **Getting Out** - If your number matches the computer's, you're OUT!
5. **Two Innings** - Both sides bat once, highest score wins

## Features
- Random computer inputs for fair gameplay
- Input validation throughout
- Two innings with target chasing

## Built With
- Python 3
- `random` module
